# Skill: Build Explainable Payout

Objective:
Make every payout readable, defensible, and easy to demo.

Tasks:
- Reuse current claims, finance, wallet, and admin simulation pathways.
- Build a readable payout ledger showing:
  - trigger crossed
  - event window
  - affected hours estimate
  - active weekly plan
  - coverage cap
  - reserve guard
  - final payout
