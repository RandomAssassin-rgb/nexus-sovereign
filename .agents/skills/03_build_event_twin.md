# Skill: Build Event Twin

Objective:
Turn a disruption into a live insurer-readable event object.

Tasks:
- Create event_instances and event_exposure_snapshots.
- Trigger Event Twin creation from simulation or auto-trigger jobs.
- Attach:
  - zone footprint
  - exposed cohort
  - projected claims
  - projected payout range
  - reserve impact
  - expected loss ratio
  - fraud distribution
- Feed output into admin dashboards.
