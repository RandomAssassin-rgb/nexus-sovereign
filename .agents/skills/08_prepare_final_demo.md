# Skill: Prepare Final Demo

Objective:
Prepare a 5-minute final demo with one unforgettable event story.

Preferred flow:
1. Worker already onboarded and protected.
2. Event is triggered in admin.
3. Event Twin opens with exposure and reserve impact.
4. One valid worker is auto-approved.
5. One suspicious worker is held.
6. Valid payout lands instantly.
7. Audit pack is shown briefly.
8. Admin dashboard updates event loss ratio and reserve position.
